Cinnamoroll Adventure

A 2D treasure hunting game for iOS devices.

Cinnamoroll Adventure takes players on a journey through a whimsical forest maze in search of a hidden treasure box. With the help of the beloved character Cinnamoroll, players must navigate through the maze, evade lurking monsters (cats), and ultimately uncover the treasure box's whereabouts.

Developement

VIEW GITHUB REPO

The initial inspiration and foundation for Cinnamoroll Adventure were sourced from Kodeco's tutorial on Introduction to the SpriteKit Scene Editor, which provided valuable insights and resources for implementing key game mechanics and features.

The Design Process:

  1. Conceptualization: Developing the initial concept and storyline for the game, outlining the objectives, challenges, and characters.
  2. Prototyping: Creating rough sketches and mockups to visualize the game mechanics, maze layouts, and character designs.
  3. Development: Writing the game using Swift programming language and the SpriteKit framework, leveraging its capabilities to create dynamic 2D graphics, animations, and interactions.
  4. Testing and Iteration: Conducting playtesting sessions to gather feedback, identify bugs, and fine-tune gameplay mechanics for optimal user experience.

Gameplay / Design

Maze Exploration

  • Players navigate through a maze-like forest environment, searching for the elusive treasure box.

Monster Avoidance

  • Cats roam the forest maze as obstacles, requiring players to evade them to avoid being caught.

Treasure Hunt

  • The key to unlocking the treasure box is hidden somewhere within the forest, adding an element of discovery and challenge.

Winning Objective

  • Successfully finding the key, unlocking the treasure box, and escaping the maze marks the completion of the game.
